# Support

This is a personal open-source project maintained on a best-effort basis.

- Questions and ideas: open a [discussion](https://github.com/zkoppert/babysit-prs/discussions) or an issue.
- Bugs: open an [issue](https://github.com/zkoppert/babysit-prs/issues) with your OS, `gh --version`, and the command you ran.

There is no service-level agreement. Pull requests are welcome (see `CONTRIBUTING.md`).
